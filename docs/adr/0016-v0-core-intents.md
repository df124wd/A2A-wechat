# v0 core intents

The v0 protocol should define ten core intents: `request`, `delegate`, `response`, `tool_call`, `tool_result`, `notify`, `refusal`, `error`, `approval_request`, and `approval_result`. This gives the first protocol version enough vocabulary for agent delegation, tool invocation, failures, refusals, status updates, and human approval without making every interaction an open-ended message type.
